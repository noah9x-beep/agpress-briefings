Morning Release Roundup

Friday, May 8, 2026
 
A note on the new look. The Release Roundup is now designed in Fraunces, a serif from Google Fonts. Gmail blocks external web fonts, so what you’re seeing in this email is Georgia (a fallback font that’s available on every device). If you’d like future briefings to render in Fraunces as intended, install the font on your devices — it’s free at fonts.google.com/specimen/Fraunces. No installation? You’ll keep seeing Georgia — no layouts break either way.
